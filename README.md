# VCFGenerator
A CLI tool for generating .vcf files from text files. Created to convert a list of phone numbers from a text file to vcf to import it in bulk on a phone. Useful for Snapchat OSINT. 

## How to get started: 
 - Simply download or clone the repo.
 - Navigate to the folder containing main.py
 - Run `python3 main.py -i your_list_of_numbers.txt -o "your_output_vcf_file.vcf". No external libraries required.
